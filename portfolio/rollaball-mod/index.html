Rollaball-Mod

Overview

Rollaball-Mod is a modded version of the original game created by a professor in the Advaced Computer Center for the Arts and Design department at The Ohio State University. Rollaball-Mod was built using Unity3D. In this game, the player is trapped in a labyrinth maze, and must roll the ball throughout the level and collect the items. The user must collect all items before unlocking one final item to win the game. If the user comes across a yellow item, special abilities are awarded to help the user collect more items. Once the user collects all items, there are animals released in the labyrinth level attempting to attack the player. If successful, the player loses and has to start over, but, if the user manages to get away and return back to home base to collect the final item, the user takes home the victory.

An alpha web version of the game is for download here.
